# What the hell
**WTH** is a client node-commandline who provides to help you with unknown files.

## Installation
You must use **npm**
```bash
	sudo npm install -g wth-cli
```

## Usage

```bash
	wth <filename with extension>
```

### Example 
This line
```bash
	wth index.js
```
